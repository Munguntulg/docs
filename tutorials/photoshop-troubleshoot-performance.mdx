## Түгээмэл асуудлууд
- Scratch disk is full
- GPU/graphics driver асуудал
- Performance/lag (маш их layers, том файлууд)

## Шийдлүүд
- Scratch disk-ийг цэвэрлэх, нэмэлт диск сонгох (`Preferences → Scratch Disks`)
- GPU драйверыг шинэчилнэ
- Photoshop-ыг `Preferences → Performance`-аас memory болон cache settings тохируулах.
