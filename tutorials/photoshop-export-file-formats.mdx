## Товч
Файлыг ямар зориулалтаар экспортолж байгаагаас хамааран формат сонгоно (web, print, archive).

## Гол форматууд
- PSD — editable source, layers хадгална
- JPEG — хэвлэх/web фото (lossy)
- PNG — ил тод байдал шаардлагатай web-график
- TIFF — өндөр чанартай хэвлэх
- SVG — вектор график (Photoshop-оор хязгаарлагдмал)

## Export арга
- `File → Export → Export As...` эсвэл `File → Save As...` ашиглана.
