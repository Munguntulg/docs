## Товчлолын товч жагсаалт (macOS / Windows)
- Буцаах (Undo): `Cmd+Z` / `Ctrl+Z`
- Давхаргыг дубльдах (Duplicate layer): `Cmd+J` / `Ctrl+J`
- Шинэ давхарга: `Shift+Cmd+N` / `Shift+Ctrl+N`
- All selections: `Cmd+A` / `Ctrl+A`
- Copy: `Cmd+C` / `Ctrl+C`
- Paste: `Cmd+V` / `Ctrl+V`

## Бүх товчлолыг харах
`Edit → Keyboard Shortcuts` эсвэл Adobe-ийн албан ёсны жагсаалтаас дэлгэрэнгүй үзнэ: https://helpx.adobe.com/photoshop/desktop/get-started/settings-and-preferences/view-keyboard-shortcuts.html
