## Товчлолын товч жагсаалт (macOS / Windows)
- Буцаах (Undo): `Cmd+Z` / `Ctrl+Z`
- Давхаргыг дубльдах (Duplicate layer): `Cmd+J` / `Ctrl+J`
- Шинэ давхарга: `Shift+Cmd+N` / `Shift+Ctrl+N`
- All selections: `Cmd+A` / `Ctrl+A`
- Copy: `Cmd+C` / `Ctrl+C`
- Paste: `Cmd+V` / `Ctrl+V`

## Нэмэлт товчлолууд (Tools, Navigation, Layers)
- Move tool: `V`
- Rectangular/Elliptical Marquee: `M`
- Lasso tool: `L`
- Quick Selection / Magic Wand: `W`
- Crop tool: `C`
- Eyedropper (color pick): `I`
- Brush tool: `B`
- Eraser tool: `E`
- Gradient / Paint Bucket: `G`
- Pen tool: `P`
- Text tool: `T`

- Free Transform: `Cmd+T` / `Ctrl+T`
- Fit on Screen: `Cmd+0` / `Ctrl+0`
- Actual Pixels (100%): `Cmd+1` / `Ctrl+1`
- Zoom In / Zoom Out: `Cmd++` / `Ctrl++` and `Cmd+-` / `Ctrl+-`
- Hand (pan) while working: hold `Space` (press and drag)
- Toggle panels: `Tab`
- Show/Hide Rulers: `Cmd+R` / `Ctrl+R`

- Deselect: `Cmd+D` / `Ctrl+D`
- Reselect: `Shift+Cmd+D` / `Shift+Ctrl+D`
- Inverse selection: `Shift+Cmd+I` / `Shift+Ctrl+I`

- Undo / Step Backward: `Cmd+Z` / `Ctrl+Z` (toggle) or `Cmd+Option+Z` / `Ctrl+Alt+Z` (step backward multiple)
- Step Forward (redo steps): `Shift+Cmd+Z` / `Shift+Ctrl+Z`

- New layer: `Shift+Cmd+N` / `Shift+Ctrl+N`
- Duplicate layer: `Cmd+J` / `Ctrl+J`
- Merge down: `Cmd+E` / `Ctrl+E`
- Merge visible: `Shift+Cmd+E` / `Shift+Ctrl+E`
- Group layers: `Cmd+G` / `Ctrl+G`
- Ungroup: `Shift+Cmd+G` / `Shift+Ctrl+G`

- Fill (Fill dialog): `Shift+F5`
- Switch foreground/background colors: `X`
- Reset foreground/background to default (black/white): `D`

Эдгээр товчлолууд нь Photoshop дээр хамгийн их ашиглагддаг бөгөөд таны ажиллах хурдыг ихээхэн сайжруулна. Хэрэв та тус бүрийн товчлолын тайлбар эсвэл нэмэлт бичлэг хэрэгтэй бол хэлнэ үү.
