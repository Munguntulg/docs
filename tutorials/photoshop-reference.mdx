## Crop Tool
| Property | Description |
| --- | --- |
| Shortcut | `C` |
| Purpose | Crop and remove parts of the image |
| Location | Toolbar |

## Brightness / Contrast
| Property | Description |
| --- | --- |
| Menu | `Image → Adjustments → Brightness/Contrast` |
| Brightness | Adjust overall lightness |
| Contrast | Adjust difference between lights and shadows |

## File Formats
| Format | Use case |
| --- | --- |
| JPG | Photographic images for web (lossy) |
| PNG | Images with transparency |
| PSD | Photoshop source file (layers preserved) |

## Shortcuts
- Undo: `Cmd+Z` / `Ctrl+Z`
- Duplicate layer: `Cmd+J` / `Ctrl+J`
- New layer: `Shift+Cmd+N` / `Shift+Ctrl+N`
